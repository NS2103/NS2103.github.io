Projects

Project 1: Blood Glucose Level Prediction

Developed a predictive model using Support Vector Regression to predict blood glucose levels based on previous day's data. This project aims to develop a predictive model to estimate the risk of Hypoglycemia i.e. sudden drop of blood glucose level below critical values in body. I did this project in collaboration with the department of Biological Science and Biotechnology at IIT Delhi.

Project 2: Prediction of Curie Temperature using Gaussian process Regression

This project was done to fullfil the requirement of a course, Materials Modelling. Out task was to create a model that will predict the curie temperature for the less-explored materials using the features such as lattice parameters. From the reference of research paper used, Gaussian Regression Process was used as a model to achieve the results.
